# Landing Page de Vendas

Este projeto ja esta pronto para publicacao gratuita como site estatico.

## Opcao recomendada: Vercel

1. Crie uma conta em [vercel.com](https://vercel.com/).
2. Envie esta pasta para um repositorio no GitHub.
3. Na Vercel, clique em `Add New -> Project`.
4. Importe o repositorio deste projeto.
5. A Vercel vai detectar automaticamente que este e um site estatico.
6. Clique em `Deploy`.
7. Ao final, a plataforma vai gerar um link publico gratuito.

## Estrutura atual

- `index.html`: pagina principal
- `styles.css`: estilos da landing page
- `vercel.json`: configuracao de deploy estatico

## Observacoes

- Todos os botoes de compra ja apontam para `https://pay.kiwify.com.br/JuglQmA`.
- Se quiser, depois voce pode conectar um dominio proprio na Vercel.
- Tambem e possivel publicar gratuitamente no Netlify ou GitHub Pages, mas a Vercel costuma ser a rota mais simples.
